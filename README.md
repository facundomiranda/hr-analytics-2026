# HR Analytics SQL Project

## Overview

This project explores employee workforce data using SQL to uncover insights related to employee attrition, compensation, demographics, tenure, and departmental performance.

Using a dataset inspired by the IBM HR Analytics Employee Attrition dataset, I analyzed key HR metrics and answered common business questions that HR teams and managers use to improve employee retention and workforce planning.

---

## Objectives

The goal of this project was to practice SQL in a realistic business environment by answering questions such as:

* What is the company's employee attrition rate?
* Which departments experience the highest turnover?
* How does salary differ between employees who stay and those who leave?
* Which age groups are most likely to leave the company?
* What is the average tenure of employees across departments?
* Which job roles have the highest attrition rates?
* How does performance relate to employee retention?
